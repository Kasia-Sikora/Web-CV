Description:

Project was generated with Angular CLI version 9.1.7.

The project was created in order to deepen knowledge in CSS and HTML.

The main assumption was to create a CV page that would contain all the necessary information about me. I think this is a more convenient and enjoyable way to view a resume than raw PDF.

My goals:

  • Expanding knowledge in the field of CSS and HTML:
    • RWD
    • User Friendly interface
    • Design pleasing to the eye
  • Practice basic knowledge about angular:
    • Page build in SPA (Single Page Application)
    • Animations
  • Server support:
    • Hosting a website on a private server
